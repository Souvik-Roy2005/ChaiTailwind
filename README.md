# ☕ ChaiTailwind

> Build your own utility-first CSS engine using JavaScript.

ChaiTailwind is a lightweight client-side styling engine inspired by utility-first frameworks like Tailwind CSS. Instead of writing CSS, you write class names, and JavaScript dynamically converts them into inline styles.

---

## 📦 GitHub Repository
👉 [[Link](https://github.com/Souvik-Roy2005/ChaiTailwind)]

---

## ✨ Features

- ⚡ Utility-first class system (`chai-*`)
- 🧠 Smart class parsing engine
- 🎨 Supports common styling utilities:
  - Spacing (padding, margin)
  - Colors (background, text)
  - Typography (font size, alignment)
  - Borders & radius
  - Flexbox utilities
- 🔄 Automatically applies styles on page load
- 👀 Supports dynamically added elements (MutationObserver)
- 🧩 Modular and scalable architecture
